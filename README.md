## 文件列表 / File List

注：弃用的脚本不会列出，还有一些懒得列，自己去探索吧

Note: Deprecated scripts are not listed, and some others have been omitted. Feel free to explore on your own.

* HumbleChoiceKeyTools/HKCT.user.js
  * HB 月包脚本，支持查看锁区信息以及只选不刮等功能
  * Humble Choice script, supports checking region lock information and features like "choose without revealing".
* HumbleGamesCollection/HGCT.user.js
  * Humble Games Collection 免客户端下载脚本
  * Humble Games Collection script for downloading without the client.
* acfun/autospeedup.user.js
  * 添加其他倍速
  * Adds additional playback speeds for acfun.
* bilibili
  * bililivecodefetcher.user.js
    * 不知道 B 站什么时候加的非要 ~~100~~ 5000 粉才能用 OBS 开直播，直接脚本秒了，迟早药丸
    * I don't know when Bilibili started requiring ~~100~~ 5000 followers to stream with OBS. This script tried to bypasses it.
* bilibili_b2a/b2a.user.js
  * BV 号转原来的数字编号
  * Converts BV numbers back to the original AV numbers for bilibili.
* coding
  * 一些原来放在 Coding 的脚本，现在 Coding 的公开仓库非常拉，于是移动过来，基本上不会更新
  * guidefix.user.js
    * Steam 指南的语言恢复多选框
    * Restores the language multi-selection checkbox for Steam Guides.
  * pageautomute.user.js
    * 网页失焦自动静音
    * Automatically mutes the page when it loses focus.
  * steamgifting.user.js
    * 解除不能赠送库存礼物给已拥有的好友
    * Removes the restriction on gifting inventory items to friends who already own the game.
* enhancedsteamcommunity-fork
  * esc.user.js
    * [Enhanced Steam Community](https://greasyfork.org/scripts/954-enhanced-steam-community) 个人修复版
    * Personal fixed version of [Enhanced Steam Community](https://greasyfork.org/scripts/954-enhanced-steam-community).
* h5player
  * CCTV.user.js
    * 替换官方播放器为 ArtPlayer，支持海外选手直接播放，绕过区域限制
    * Replaces the official player with ArtPlayer, allowing overseas users to play directly, bypassing region restrictions.
  * videospeedup.user.js
    * 给我一些常看动漫的站点提供快捷键加速播放（我习惯 4x
    * Provides keyboard shortcuts for playback speed on some anime sites (I'm used to 4x).
    * Shift+12345678
      * 直接设置视频为指定倍速
      * Directly sets the video to the specified speed.
    * Shift+<>
      * 快进快退30秒
      * Fast forward/rewind 30 seconds.
    * f
      * 切换全屏
      * Toggle fullscreen.
    * -=
      * **按住**以当前倍速的一半/两倍速度播放视频
      * **Hold down** to play at half/double the current speed.
* other
  * blocksw.user.js
    * 尝试禁用所有网站的 Service Worker，可能导致部分网站无法正常加载
    * Attempts to disable Service Workers on all sites, which may cause some sites to not load properly.
  * neteasecgadskipper.user.js
    * 网易云游戏广告快速跳过
    * Quickly skips ads on NetEase Cloud Gaming.
  * stcnmdeditor.user.js
    * 其乐发帖编辑器中的 markdown 助手
    * A markdown helper for the Keylol post editor.
  * tunnelbeardumper.user.js
    * 将 TunnelBear 的节点提取为浏览器直接可用的 https 代理
    * Extracts TunnelBear nodes as HTTPS proxies that can be used directly by the browser.
* steam
  * curatordel.user.js
    * 给 Steam 鉴赏家已发表评测增加快捷删除按钮
    * Adds a quick delete button for published reviews for Steam Curators.
  * marketshowmore.user.js
    * 让 Steam 市场的价格列表显示更多
    * Makes the Steam Market price list show more items.
  * reviewtools.user.js
    * 给 Steam 游戏评测增加一些编辑工具，增强使用体验
    * Adds some editing tools to Steam game reviews to enhance the user experience.
  * steamnotificationsforphone.user.js
    * 手机端看 Steam 通知改进
    * Improves viewing Steam notifications on mobile.
  * steamcyberfamilynotify.user.js
    * 赛博家庭成员（除了自己之外）有新游戏时弹出通知
    * Shows a notification when a "cyber family member" (other than yourself) gets a new game on Steam.
